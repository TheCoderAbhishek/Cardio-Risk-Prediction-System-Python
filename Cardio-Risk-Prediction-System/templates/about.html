What is CRPS?





Cardio Risk Prediction System is a user-friendly web-based application which predicts heart disease using Logistic Regression a Machine Learning Algorithm. How can you use CRPS?





Anyone with a device having browser and internet can easily use Cardio Risk Prediction System. If you want to predict heart disease using CRPS, all you need is to enter accurate value of 13 different attributes that system asks like age, sex, blood pressure and so on. CRPS will then return the chances of having heart disease in probability which resides from 0-1. In addition to that, if you want to save data for future purpose and want guidance on what to do next, you can simply sign up to the system which provides all those features.
